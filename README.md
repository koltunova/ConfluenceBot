# ConfluenceBot

Набор скриптов и утилит для работы с API Confluence. Цель проекта—организовать документы при помощи ИИ и автоматизировать перенос страниц между пространствами.

Подробнее об архитектуре и планах см. [docs/ARCHITECTURE.md](docs/ARCHITECTURE.md).
Краткое описание консольного интерфейса расположено в [docs/CONSOLE_UI.md](docs/CONSOLE_UI.md).
